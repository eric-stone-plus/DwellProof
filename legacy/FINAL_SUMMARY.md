## 三、长春卖房核心结论

### 3.1 市场状态

```
Regime: R2下行(二手环比99.8, 仍在跌)
峰值: 2019-04 同比111.0
谷底: 2023-04 同比93.0
当前: 2026-04 同比95.9(回升2.9)
```

### 3.2 卖房策略

```
核心区(南关/朝阳): 挂牌价=成交价×0.95, 1-3月成交
非核心区(宽城/二道): 挂牌价=成交价×0.90, 2-4月成交
新区(净月): 挂牌价=成交价×0.88, 3-6月成交
一汽配套(汽开): 挂牌价=成交价×0.93, 2-3月成交
```

### 3.3 多套房策略

```
先卖最差的(远郊/老破小) → 挂牌价×0.88
再卖非核心区(宽城/二道) → 挂牌价×0.90
最后卖核心区(南关/朝阳) → 挂牌价×0.95
保留: 学区房/核心区优质房(最抗跌)
卖不出去: 出租(租售比3.3%, 全国最高)
```

### 3.4 时间窗口

```
最佳: 3-4月挂盘, 6月前必须成交
次选: 5-6月挂盘, 9月前成交
避免: 10-12月(淡季)
如果可以等: 等环比转正(预计2026Q3-Q4)
```

### 7.2 管理房产

```python
from housealice_uha import UnifiedHousingAccount

uha = UnifiedHousingAccount("我的房产")

# 添加房产

    city="长春", district="南关", address="XXX",
    buy_price=80, buy_date="2018-06-01", area=90,
)

# 查看报告
print(uha.summary())
```
