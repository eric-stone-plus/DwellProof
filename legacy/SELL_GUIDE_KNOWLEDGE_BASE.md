## 一、时机判断

### Q1: 现在该不该卖？
**看环比，不看同比。**
- 环比>100.0 = 市场在涨，可以卖
- 环比99.8-100.0 = 接近底部，再等等
- 环比<99.8 = 还在跌，别急

当前(2026.04): 上海100.7/北京100.4/广州100.2/武汉100.1 → 一线城市窗口已开。

### Q2: 一年中什么时候卖最好？
**所有城市都是3月最强（小阳春）。**
- 最佳: 3月 > 4月 > 5月
- 最差: 10月 < 11月 < 12月
- 原因: 春节后需求集中释放 + 学区房3月截止

### Q3: 同比跌了7%，是不是亏了？
**同比跌≠你亏了。** 同比是跟去年同月比。你买房时的价格才是你的成本。
- 如果你2020年买的，同比跌7%但你可能还赚20%
- 如果你2021年高位买的，确实亏了
- 关键: 看你的买入价 vs 当前挂牌价，不是看同比

## 二、定价策略

### Q4: 挂牌价怎么定？
```
挂牌价 = 同小区最近3月成交均价 × 0.97

留3%的"谈判空间"，让买家觉得砍到了价。
```

### Q5: 议价空间留多少？
- 市场好(环比>100.3): 留3-5%
- 市场一般(环比100.0-100.3): 留5-8%
- 市场差(环比<100.0): 留8-12%

### Q6: 挂多久没卖就该降价？
- 1个月没带看 → 检查房源描述/照片
- 2个月没成交 → 降价3%
- 3个月没成交 → 降价5%
- 6个月没成交 → 降价8-10%或换中介

## 三、买家心理

### Q7: 买家在等什么？
- 等房价继续跌（"再等等能便宜"）
- 等利率继续降（"月供能再少点"）
- 等政策放松（"限购取消了再买"）
- **破解**: 让买家觉得"现在不买以后更贵"

### Q8: 怎么让房子更好卖？
- **价格**: 比同小区低2-3%就是最大的竞争力
- **照片**: 专业拍摄，明亮干净
- **看房**: 保持整洁，通风采光
- **信息**: 详细描述（楼层/朝向/装修/学区/地铁）
- **时机**: 周末带看量最高

## 四、税费计算

### Q9: 卖房要交什么税？
```
增值税: 满2年免征，不满2年全额5.3%
个税: 满5年唯一免征，否则差额20%或全额1%
契税: 买方交，但影响买方出价
中介费: 1-2%（可谈）
```

### Q10: "满五唯一"省多少？
假设房子赚了100万:
- 不满五唯一: 个税 = 100万×20% = 20万
- 满五唯一: 个税 = 0
- **省20万** → 这是买家最在意的

## 五、市场信号

### Q11: 怎么判断市场底部？
**上海二手房环比是最可靠的领先指标。**
- 上海触底 → 全国2-5月后跟底
- 上海环比转正 → 全国3-6月后跟正
- 当前: 上海2026.03转正(100.4) → 全国5-8月跟正

### Q12: 怎么判断市场见顶？
- 成交量先于价格见顶
- 挂牌量突然增加
- 环比从>100.5回落到<100.3
- 中介开始主动找你降价

### Q13: 政策对房价的影响？
- LPR降 → 利好，但传导需6-12月
- 限购放松 → 利好一线，对三线影响小
- 首付降低 → 利好刚需，对改善影响有限
- **政策不是万能的** → 2024-2025的组合拳只让一线触底，三线仍在跌

## 六、特殊情况

### Q14: 房子有贷款怎么卖？
- **转按揭**: 买家接手你的贷款（需银行同意）
- **提前还款**: 用买家首付还清贷款 → 解押 → 过户
- **带押过户**: 部分城市已支持，不用先还贷

### Q15: 卖房换房要注意什么？
- **先卖后买**: 安全，但可能错过买入窗口
- **先买后卖**: 有风险，但如果市场好可以两头赚
- **建议**: 当前市场先卖后买，锁定卖出价再找新房

### Q16: 要不要等市场回暖再卖？
**看你的资金需求。**
- 如果不急: 等环比稳定>100.3再卖（预计2026Q3-Q4）
- 如果急用钱: 现在环比100.2已经是窗口，可以出手
- 如果等得起: 2027年同比可能回到95+，价格更高

## 七、城市特供

### 广州
- 天河/越秀最保值，番禺/黄埔供应大
- 3月小阳春特别明显（环比均值100.60）
- 珠江新城/天河北/东风东 = 核心资产

### 上海
- 内环>中环>外环>郊环
- 学区房政策变化大，关注最新划片
- 二手市场最活跃，流动性最好

### 北京
- 东西城学区房最抗跌
- 五环外跌幅最大
- 限竞房/共有产权房冲击二手市场

### 武汉
- 全国跌幅最深(同比90.6)
- 光谷/汉口中心相对好
- 远城区(江夏/黄陂)风险最大
