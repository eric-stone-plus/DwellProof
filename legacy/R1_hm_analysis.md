## 四、政策传导分析

### LPR传导效果

LPR 5年期 3.5%（历史最低）→ 首套房贷利率 ≈ 3.2%

| 时间 | LPR 5Y | 100万/30年月供 | 较2021高点节省 |
|------|--------|--------------|--------------|
| 2021.12 (高点) | 4.65% | ¥5,155 | - |
| 2024.10 | 3.60% | ¥4,546 | ¥609/月 |
| 2026.05 | 3.50% | ¥4,490 | ¥665/月 |

**传导不畅的原因:**
1. 房价在跌 → 购房者等跌到位
2. 收入预期不稳 → 加杠杆意愿低
3. 二手房挂牌量高位 → 卖方竞争激烈

### 政策 vs 市场的博弈

```
政策端: LPR↓ + 限购放松 + 公积金↑ + 首付↓
市场端: 房价↓ + 挂牌量↑ + 成交量↑(以价换量) + 观望情绪浓

结论: 政策在"托"，市场在"磨"。一线城市先企稳，二三线仍在寻底。
```
