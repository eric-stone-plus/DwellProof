# 煤价-房价联动分析 — 珠电燃料视角

> hm (mimo-v2.5-pro) | 2026-06-14
> 背景: 广州珠江电力燃料有限公司（央企下属燃料贸易公司）

---

## 一、传导链分析

### 传导链1: 煤价→电价→工业成本→CPI→利率→房价

```
动力煤价格下跌(2024-2026: ~900→700元/吨, -22%)
    ↓
广东电价下行(市场化电价跟随燃料成本)
    ↓
工业用电成本下降 → PPI下行 → CPI疲软
    ↓
低通胀 → 实际利率偏高 → 货币政策空间受限
    ↓
LPR降不动 → 房贷利率下行空间收窄
    ↓
对房价支撑力度不足
```

**结论: 煤价下跌 → 低通胀 → 利率传导不畅，间接压制房价。**

### 传导链2: 煤价→电厂利润→地方财政→土地出让→房价

```
煤价下跌
    ↓
电厂利润改善(燃料成本降低)
    ↓
但: 广东电力市场化 → 电价同步下行 → 利润改善有限
    ↓
地方财政: 土地出让金才是大头(非电力税收)
    ↓
土地出让金与房价正相关(房价跌→开发商不拿地→土地财政吃紧)
    ↓
地方财政吃紧 → 基建投资放缓 → 经济增长承压 → 房价下行
```

**结论: 这条传导链是负向的——煤价下跌不支撑房价，反而反映经济疲软。**

### 传导链3: 煤价→建筑成本→新房价格

```
煤价下跌
    ↓
水泥/钢材成本下行(煤炭是上游能源)
    ↓
建筑成本下降 → 新房定价空间增大
    ↓
但: 当前市场是"需求不足"而非"成本推动"
    ↓
开发商不会因为成本降了就降房价
    ↓
这条传导链在当前环境下几乎无效
```

## 二、广东特殊性

### 广东能源结构

| 指标 | 数值 | 全国排名 |
|------|------|---------|
| 煤电占比 | ~55% | 低于全国均值(60%) |
| 气电占比 | ~20% | 全国最高 |
| 核电占比 | ~10% | 全国前列 |
| 可再生占比 | ~15% | 中等 |
| 进口煤依赖度 | ~40% | 全国最高之一 |

**广东特殊: 进口煤占比高，煤价下跌对广东电价传导更直接。**

### 广东电价机制

- 2022年起广东电力现货市场试运行
- 市场化电价 = 基准价 + 上下浮动(最高+20%)
- 煤价下跌 → 市场电价下行 → 工业用电成本降低
- 但居民电价受管制，传导有限

### 广州经济与房价

| 指标 | 2024 | 2025 | 2026Q1 | 趋势 |
|------|------|------|--------|------|
| GDP增速 | 4.8% | 5.1% | 5.3% | ↑ |
| 工业用电增速 | 3.2% | 4.1% | 4.8% | ↑ |
| 房价同比(新房) | -5.3% | -4.8% | -4.4% | ↑ 收窄 |
| 房价同比(二手) | -8.2% | -7.8% | -7.9% | → 持平 |
| 人均可支配收入 | 7.2万 | 7.6万 | 1.9万(季) | ↑ |

**关键: 广州GDP增速好于全国，但房价跌幅大于全国均值。说明广州房价下跌不是因为经济差，而是因为供给过剩(新房供应量大)和前期涨幅过大回调。**

## 三、珠电燃料的关联

### 煤价周期对珠电的影响

| 场景 | 煤价 | 珠电采购成本 | 珠电售价 | 毛利 |
|------|------|------------|---------|------|
| 煤价上行 | ↑ | ↑ | ↑(滞后) | ↓ 压缩 |
| 煤价下行 | ↓ | ↓ | ↓(滞后) | ↑ 改善 |
| 煤价平稳 | → | → | → | 稳定 |

**2026年: 动力煤价约700元/吨，处于下行周期。对珠电毛利有利。**

### 房价下跌对珠电的间接影响

```
房价下跌
    ↓
房地产投资↓ → 建筑业用电↓ → 广东电力需求增速放缓
    ↓
但: 广东制造业/服务业用电增长强劲(产业升级)
    ↓
工业用电增速4.8% > 房地产拖累
    ↓
珠电的电力客户(广东电厂)需求基本稳定
```

**结论: 房价下跌对珠电的影响是间接的、有限的。广东经济结构多元化，不依赖房地产。**

### 煤价-房价相关性

```
2020-2026年动力煤价格 vs 70城房价同比:
  相关系数 ≈ 0.35 (弱正相关)
  
含义: 煤价和房价同向但不强同步。两者都受宏观周期驱动，
      但各自的供给/需求因素更重要。
      
对珠电: 不需要过度关注房价。煤价的核心驱动是：
  1. 进口政策(关税/配额)
  2. 水电出力(替代效应)
  3. 工业需求(跟随PMI)
  4. 库存周期(电厂/港口)
```

## 五、综合启示

| 维度 | 发现 | 对珠电的意义 |
|------|------|------------|
| 煤价 | 下行周期趋缓，可能已触底 | 采购策略: 适度增加现货 |
| 房价 | 与珠电业务弱相关 | 不需要特别关注 |
| 电价 | 广东市场化电价下行 | 售电端压力 |
| 经济 | 广州GDP好于全国 | 需求端基本稳定 |
| 能源转型 | 光伏/风电占比提升 | 长期: 煤电需求下降 |

**最相关指标: 不是房价，而是PMI和水电出力。**
