## 四、卖房策略

### 4.1 当前市场判断

```
信号: 二手环比99.8 (仍在下跌, 但跌幅收窄)
判断: 🟠 不适合卖(还在跌)
建议: 等环比转正(>100)再卖
预计: 2026下半年可能触底(如果趋势延续)
```

### 4.2 如果必须现在卖

**定价策略:**
```
挂牌价 = 同小区最近3月成交价 × 0.92 (留8%议价空间)

原因:
- 长春市场弱, 买家议价能力强
- 二手环比99.8, 每月都在跌
- 不降价基本卖不出去
```

**具体操作:**
1. 查贝壳/链家同小区最近3月成交价
2. 挂牌价 = 成交价 × 0.92
3. 挂盘后1个月无带看 → 降价3%
4. 挂盘后2个月无成交 → 再降3%
5. 挂盘后3个月无成交 → 降到底价或放弃

**最佳卖房时间:**
```
3-4月: 小阳春(环比均值最高)
5-6月: 还可以
7-9月: 淡季
10-12月: 最差(别挂盘)
```

### 4.3 如果可以等

**等待信号:**
```
1. 长春二手环比连续2月>100 → 触底信号
2. 上海二手环比>100.5 → 全国回暖信号(领先3月)
3. 沈阳二手环比>100.3 → 东北回暖信号(领先1月)
```

**预计时间:**
```
如果趋势延续(环比从99.8缓慢回升):
- 触底: 2026Q3-Q4
- 环比转正: 2027Q1
- 同比转正: 2028-2029
```

### 4.4 多套房策略

```
如果有多套, 建议:
1. 先卖最差的(远郊/老破小/配套弱)
2. 保留最好的(核心区/学区/配套好)
3. 分批卖(不要一次全挂)
4. 每套独立定价(不要统一定价)
```
