# 🏠 二手房交易全流程 + 时间线 + 费用清单

## 一、卖房全流程时间线

```
第1周: 准备阶段
├── 查产权(不动产登记中心)
├── 查同小区成交价(贝壳/链家)
├── 确定挂牌价(成交价×0.97)
├── 准备房产证+身份证+户口本
└── 选中介(2-3家同时挂)

第2周: 挂盘阶段
├── 拍照(专业摄影)
├── 写房源描述(详细+卖点)
├── 多平台挂盘(贝壳/链家/58)
└── 等待带看

第3-4周: 带看阶段
├── 保持房屋整洁
├── 灵活安排看房时间
├── 收集买家反馈
└── 调整挂牌价(如有必要)

第5-8周: 谈判阶段
├── 接受/还价
├── 确定付款方式
├── 确定过户时间
├── 签意向书+收定金
└── 如6周无成交→考虑降价

第9-12周: 签约过户
├── 签正式合同
├── 网签(房管局备案)
├── 买方申请贷款(1-2周)
├── 过户(房管局)
├── 买方放款(银行)
└── 收到尾款

第13-14周: 交房
├── 交钥匙
├── 结清水电气物业费
├── 迁出户口
└── 交房完成

总时间: 约3个月(顺利)到6个月(不顺利)
```

---

## 二、买房全流程时间线

```
第1-2周: 准备阶段
├── 查征信(银行)
├── 预审贷款(银行)
├── 确定预算
├── 选区域+看房
└── 选中介

第3-6周: 看房阶段
├── 多看几套(至少10套)
├── 不同时间段看(上午/下午/晚上)
├── 了解小区环境+邻居
├── 查学校+地铁+配套
└── 缩小范围

第7-8周: 谈判签约
├── 出价
├── 谈判
├── 签意向书+付定金(5-10万)
├── 签正式合同
└── 付首付

第9-12周: 贷款过户
├── 网签(房管局)
├── 评估(银行)
├── 审批贷款(1-2周)
├── 过户(1-2周)
├── 放款(1-2周)
└── 收到房产证

第13-14周: 交房验房
├── 验房(请专业验房师)
├── 交钥匙
├── 水电气过户
├── 物业过户
└── 入住

总时间: 约3个月(顺利)到6个月(不顺利)
```

## 五、关键结论

1. **卖房**: 3月挂盘最佳，6周内成交最理想
2. **买房**: 准备充分+多看+不急
3. **费用**: 卖方3-9万，买方51-56万(含首付)
4. **持有成本**: 每年13-14万(含月供)
5. **最佳时间**: 3-4月(成交量+价格双高)
