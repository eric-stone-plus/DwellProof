## 一、买房 vs 租房 10年测算

### 假设
- 房价: 300万(广州天河80㎡)
- 首付: 45万(15%)
- 贷款: 255万, 30年, 利率3.2%
- 月供: ¥11,026
- 租金: ¥5,000/月(同地段)
- 房价年涨幅: 假设不同情景

### 情景1: 房价年涨3%(乐观)
```
10年后房价: 300万 × 1.03^10 = 403万
总月供: ¥11,026 × 120 = ¥132.3万
首付: ¥45万
总投入: ¥177.3万
房产净值: 403万 - 剩余贷款(约210万) = 193万
投资回报: (193 - 177.3) / 177.3 = 8.9% (10年总)

如果租房: 租金¥5,000 × 120 = ¥60万
省下的钱(月供差额¥6,026 × 120) = ¥72.3万 理财(年化3%) = ¥83万
租房总成本: ¥60万
租房净资产: ¥83万

结论: 买房(193万) > 租房(83万) → 买房更划算
```

### 情景2: 房价年涨0%(横盘)
```
10年后房价: 300万
房产净值: 300 - 210 = 90万
投资回报: (90 - 177.3) / 177.3 = -49% (亏损)

租房净资产: ¥83万

结论: 租房(83万) ≈ 买房(90万) → 差不多
```

### 情景3: 房价年跌2%(阴跌)
```
10年后房价: 300万 × 0.98^10 = 245万
房产净值: 245 - 210 = 35万
投资回报: (35 - 177.3) / 177.3 = -80% (巨亏)

租房净资产: ¥83万

结论: 租房(83万) >> 买房(35万) → 租房更划算
```

## 五、关键结论

1. **自住**: 利率3.2%(历史最低)+房价跌了7-23% → 自住刚需现在买没问题

5. **买房时机**: 一线城市环比已转正，现在是窗口期
6. **卖房时机**: 3月最佳(小阳春)，10-11月最差
