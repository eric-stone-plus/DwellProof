## 三、综合分析

### 信贷与房价的关系

```
新增信贷↑ → 购房资金充裕 → 房价↑
新增信贷↓ → 购房资金紧张 → 房价↓
LPR↓ → 房贷成本↓ → 需求↑ → 房价↑
LPR↑ → 房贷成本↑ → 需求↓ → 房价↓
```

- 货币政策对房价中性偏利好(LPR低+首付低)
