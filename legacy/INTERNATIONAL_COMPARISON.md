## 三、房地产投资组合模型

### 3.1 分散投资策略

```
投资组合 = 核心资产(60%) + 增长资产(30%) + 机会资产(10%)

核心资产(60%): 一线核心区住宅
  - 上海内环/北京东西城/广州天河越秀
  - 特点: 保值、流动性好、抗跌
  - 预期回报: 年化2-5%

增长资产(30%): 强二线核心区住宅
  - 成都锦江/杭州上城/武汉武昌
  - 特点: 有人口流入、租金回报较高
  - 预期回报: 年化3-8%

机会资产(10%): 超跌城市抄底
  - 武汉/郑州(跌幅最深)
  - 特点: 高风险高回报
  - 预期回报: 年化-5%~+15%
```

### 3.2 各城市投资权重

| 城市 | 类型 | 权重 | 理由 |
|------|------|------|------|
| 上海 | 核心 | 20% | 最保值、流动性最好 |
| 北京 | 核心 | 15% | 政治中心、学区强 |
| 广州 | 核心 | 15% | 性价比最高一线 |
| 深圳 | 核心 | 10% | 科技中心、但租售比低 |
| 成都 | 增长 | 10% | 人口流入、租售比合理 |
| 杭州 | 增长 | 8% | 互联网中心、但房价高 |
| 武汉 | 增长 | 7% | 教育中心、跌幅深 |
| 南京 | 增长 | 5% | 长三角核心 |
| 郑州 | 机会 | 5% | 超跌、租售比合理 |
| 西安 | 机会 | 3% | 西北中心、刚触底 |
| 长春 | 机会 | 2% | 租售比最高 |

### 3.3 风险管理

| 风险 | 对冲方式 |
|------|---------|
| 房价下跌 | 分散城市+核心区 |
| 利率上升 | 固定利率贷款 |
| 流动性风险 | 保持30%现金 |
| 政策风险 | 关注政策动向 |
| 租金下降 | 选择刚需区域 |
