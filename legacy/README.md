## 新版入口

新版二手房投资证据工作台位于 `../web/`。从项目根目录启动：

```bash
cd web
npm run dev
```

只有当房源级证据门槛全部通过，并且税费、银行实际执行利率和可比成交都具有可追溯来源与有效日期时，才可进入情景计算；证据不足时必须保持 `INSUFFICIENT_EVIDENCE`。
