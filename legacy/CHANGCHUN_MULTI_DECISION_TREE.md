## 一、决策树

```
Q1: 你有几套房需要卖?
├── 1套 → 直接跳到Q4
├── 2-3套 → Q2
└── 4套以上 → Q3

Q2: 2-3套房, 先卖哪套?
├── 有远郊/老破小 → 先卖远郊/老破小
├── 有核心区+非核心区 → 先卖非核心区
├── 有学区房 → 最后卖(最抗跌)
└── 都差不多 → 先卖挂牌量最大的区域

Q3: 4套以上, 分批策略
├── 第一批(立即): 远郊/老破小/配套弱
├── 第二批(3-6月): 非核心区普通房
├── 第三批(6-12月): 核心区普通房
└── 保留: 学区房/核心区优质房

Q4: 单套房怎么卖?
├── 核心区(南关/朝阳) → 挂牌价=成交价×0.95
├── 非核心区(宽城/二道/绿园) → 挂牌价=成交价×0.90
├── 新区(净月) → 挂牌价=成交价×0.88
└── 一汽配套(汽开) → 挂牌价=成交价×0.93

Q5: 什么时候卖?
├── 必须现在卖 → 3-4月挂盘, 6月前成交
├── 可以等3个月 → 等环比转正信号
├── 可以等6个月 → 等2026Q3-Q4触底
└── 不急 → 等2027年(预计环比转正)

Q6: 卖不出去怎么办?
├── 1个月无带看 → 降价3%
├── 2个月无成交 → 再降3%
├── 3个月无成交 → 降到底价或放弃
└── 6个月无成交 → 考虑出租(租售比3.3%)
```

## 二、多套房优先级排序

### 2.1 排序原则

```
优先级 = 区域风险 × 流动性 × 持有成本

区域风险: 远郊 > 非核心区 > 核心区
流动性:   老破小 > 普通房 > 学区房
持有成本: 贷款房 > 全款房
```

### 2.2 具体排序

| 优先级 | 房产类型 | 原因 | 挂牌策略 |
|--------|---------|------|---------|
| 1(最先卖) | 远郊/老破小 | 风险最高+流动性最差 | 挂最低价 |
| 2 | 非核心区普通房 | 供应量大+需求弱 | 挂低价 |
| 3 | 核心区普通房 | 有一定需求 | 挂市场价 |
| 4(最后卖) | 学区房/核心区优质房 | 最抗跌+需求稳定 | 挂稍高价 |

## 三、具体操作计划

### 3.1 第一批(立即行动)

```
目标: 远郊/老破小/配套弱
时间: 本周挂盘
挂牌价: 成交价 × 0.88
预期成交: 2-4个月
行动:
  1. 查贝壳同小区成交价
  2. 挂牌价=成交价×0.88
  3. 挂2-3家中介
  4. 每周跟进带看量
  5. 1个月无带看→降价3%
```

### 3.2 第二批(1-2周后)

```
目标: 非核心区普通房
时间: 第一批挂盘后1-2周
挂牌价: 成交价 × 0.90
预期成交: 2-4个月
行动:
  1. 同上流程
  2. 关注第一批反馈, 调整策略
```

### 3.3 第三批(1个月后)

```
目标: 核心区普通房
时间: 第二批挂盘后1个月
挂牌价: 成交价 × 0.93-0.95
预期成交: 1-3个月
行动:
  1. 同上流程
  2. 核心区可以挂稍高价
```

### 3.4 保留

```
目标: 学区房/核心区优质房
策略: 持有
原因:
  - 学区房最抗跌(刚需)
  - 核心区优质房流动性好
  - 租售比3.3%(出租也有收益)
  - 等市场回暖再卖
```
