# 房产投资决策框架

## 编制说明
本手册系统梳理中国内地房产投资的决策方法论，覆盖城市选择、时机判断、
持有/卖出决策、置换策略。基于广州/长春房产实践和70城数据分析。

---

## 一、城市分级与选择

### 1.1 城市分级体系
```
一线：北京/上海/广州/深圳
强二线：杭州/成都/武汉/南京/苏州/西安
弱二线：长沙/郑州/合肥/青岛/大连/厦门
三线：其他省会+经济强市
四线：普通地级市
五线：县级市/县城
```

### 1.2 城市选择指标
| 指标 | 权重 | 说明 |
|------|------|------|
| 人口净流入 | 25% | 近3年常住人口变动 |
| GDP增速 | 15% | 经济活力 |
| 人均可支配收入 | 15% | 购买力 |
| 土地供应 | 15% | 供给端约束 |
| 库存去化月数 | 15% | 供需平衡 |
| 产业多样性 | 10% | 抗风险能力 |
| 交通便利度 | 5% | 城市群融入度 |

### 1.3 广州 vs 长春对比
| 维度 | 广州 | 长春 |
|------|------|------|
| 城市分级 | 一线 | 弱二线 |

## 二、时机判断

### 2.1 领先指标体系
```
领先指标（提前3-6月）：
  - 房企拿地面积
  - 房企到位资金
  - 个人按揭贷款利率
  - 土地溢价率
  - 房企新开工面积

同步指标：
  - 商品房销售面积
  - 商品房销售额
  - 70城房价指数

滞后指标：
  - 房地产投资完成额
  - 竣工面积
  - 房企库存
```

### 2.2 上海二手房价领先指标
```
上海二手住宅环比指数是全国房价领先指标（领先2-5月）
原因：
  - 上海市场最敏感，政策反应最快
  - 二手房市场化程度最高
  - 投资客占比高，信号传导快
应用：
  上海环比转正 → 2-5月后其他一线跟涨
  上海环比转负 → 2-5月后其他一线跟跌
```

### 2.3 最佳买卖月份
```
最佳卖房月：3月（学区房需求+年后购房潮）
次佳卖房月：9月（金九银十）
最差卖房月：1-2月（春节淡季）、7-8月（酷暑）

最佳买房月：1-2月（议价空间大）
次佳买房月：7-8月（淡季捡漏）
```

## 三、持有/卖出决策

### 3.1 决策矩阵
```
如果 房价预期涨幅 > 资金成本 + 机会成本 → 持有
如果 房价预期涨幅 < 资金成本 + 机会成本 → 卖出
如果 不确定 → 分批处置
```

### 3.2 持有成本计算
```
年持有成本 = 房贷利息 + 物业费 + 维修基金 + 房产税(如有) + 折旧
         = 贷款余额 × 利率 + 固定费用 + 评估折旧
```

### 3.3 机会成本
```
机会成本 = 房产市值 × 无风险收益率
         = 房产市值 × 3% (国债/大额存单)
```

### 3.4 卖出决策流程
```
1. 评估当前市值（参考近期成交）
2. 计算持有成本（年化）
3. 预测未来12月涨幅
4. 比较：预期涨幅 vs 持有成本+机会成本
5. 决策：持有/卖出/分批
```

## 四、置换策略

### 4.1 卖旧买新
```
时间窗口：先卖后买（避免两头踏空）
资金安排：卖房款作为首付
风险控制：预留6个月月供作为缓冲
```

### 4.2 跨城置换
```
低线→高线：资产升级，长期看好
高线→低线：套现离场，改善生活
注意事项：限购政策、贷款政策、税费
```

### 4.3 置换成本
```
总成本 = 卖方税费 + 买方税费 + 中介费 + 装修费 + 搬迁费
卖方税费 = 增值税(满2年免) + 个税(满5唯一免) + 中介费1-2%
买方税费 = 契税(1-3%) + 中介费1-2%
```

## 五、广州房产策略

### 5.1 区域分级
```
核心持有：天河/越秀/海珠(琶洲)
观察持有：白云/番禺(万博)/黄埔(科学城)
考虑处置：花都/从化/增城/南沙(远期)
```

### 5.2 广州市场特征
- 新房供应充足，二手房竞争激烈
- 学区房溢价明显（越秀/天河）
- 珠江新城/金融城是价值高地
- 南沙/增城远期有潜力但短期承压

### 5.3 广州操作建议
```
核心地段（天河/越秀）：持有，等市场回暖
非核心地段（花都/从化）：尽快处置，回笼资金
多套持有：集中到1-2套核心资产
```

## 六、长春房产策略

### 6.1 市场特征
- 人口净流出，需求萎缩
- 库存高企，去化周期长
- 新房降价促销频繁
- 二手房流动性差

### 6.2 长春操作建议
```
核心地段（南关/朝阳）：降价也要卖，止损优先
非核心地段：大幅降价，尽快脱手
多套持有：全部处置，回笼资金
```

### 6.3 处置策略
```
定价策略：参考近3月成交价，下浮5-10%
营销策略：多平台挂牌、中介合作、朋友圈推广
谈判策略：底价明确，灵活付款方式
时间策略：3个月内必须成交，否则继续降价
```

## 七、风险控制

### 7.1 杠杆控制
```
房贷收入比 < 50%
月供不超过家庭月收入50%
预留6个月月供作为应急资金
```

### 7.2 流动性管理
```
不要把所有资金锁在房产中
保持30-50%流动资产（股票/基金/现金）
房产投资周期长（5-10年），要有耐心
```

### 7.3 政策风险
```
限购政策：可能放松也可能加严
贷款政策：利率/首付比例变动
房产税：长期趋势，提前规划
```
